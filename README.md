# Moto Clases PHP

## Descripción

Este proyecto es un ejemplo simple de diseño de clases en PHP para representar una moto. Permite al usuario configurar propiedades como la potencia, el color y el modelo, y muestra el resultado en un modal.
## Requisitos

- Servidor web (por ejemplo, Apache)
- PHP
- Navegador web moderno

## Instrucciones de Uso

1. Clona este repositorio:

    ```bash
    git clone https://github.com/AlexisTech41/moto_clases_php.git
    ```

2. Configura tu servidor web para apuntar al directorio del proyecto.

3. Abre el navegador y accede al proyecto.

4. Completa el formulario para configurar la moto.

5. Observa la salida en el modal.
